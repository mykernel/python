要求
把嵌套的字典转化为 一层字典，实现效果如下：

源字典  {'a':{'b':1,'c':2},'d':{'e':3,'f':{g:4}}}
目标字典 {'a.b': 1, 'a.c': 2, 'd.e': 3, 'd.f.g': 4}

```python
source = {'a':{'b':1,'c':2},'d':{'e':3,'f':{'g':4}}}
```


```python
target={}
def flatmap(src,pre=''):
    for k,v in src.items():
 
        if isinstance(v,(list,tuple,set,dict)):
            flatmap(v,pre=pre+k+'.')
        else:
            target[pre+k] = v


flatmap(source)
print(target)
```

    {'a.b': 1, 'a.c': 2, 'd.e': 3, 'd.f.g': 4}
