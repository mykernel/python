给定 n 个学生的学号(从 1 到 n 编号)以及他们的考试成绩，表示为(学号，考试成绩)，请将这些学生按考试成绩降序排序，若考试成绩相同，则按学号升序排序。
 
样例
样例1

输入: array = [[2,50],[1,50],[3,100]]
输出: [[3,100],[1,50],[2,50]]
样例2

输入: array = [[2,50],[1,50],[3,50]]
输出: [[1,50],[2,50],[3,50]]

```python
array = [[2,50],[1,50],[3,100]]

sorted(sorted(array),key=lambda x:x[1],reverse=True)
```




    [[3, 100], [1, 50], [2, 50]]
