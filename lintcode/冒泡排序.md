给一组整数，按照升序排序，使用选择排序，冒泡排序，插入排序或者任何 O(n2) 的排序算法。

```python
lst = [3, 2, 1, 4, 5]
```


```python
for i in range(len(lst)-1):
    for i in range(len(lst)-1):
        if lst[i] <  lst[i+1]:
            lst[i],lst[i+1] = lst[i+1],lst[i]

print(lst)
```
