给一个整数数组，找到两个数使得他们的和等于一个给定的数 target。

你需要实现的函数twoSum需要返回这两个数的下标, 并且第一个下标小于第二个下标。注意这里下标的范围是 0 到 n-1。

```python
numbers  = [1,3,4,5,6]
target = 10

for i in range(len(numbers)):
    for j in range(i +1,len(numbers)):
        if numbers[i] + numbers[j] == target:
            print(i, j)
```
