给定一个整型数组，找出主元素，它在数组中的出现次数严格大于数组元素个数的二分之一。



你可以假设数组非空，且数组中总是存在主元素。

样例
样例 1:

输入: [1, 1, 1, 1, 2, 2, 2]
输出: 1
样例 2:

输入: [1, 1, 1, 2, 2, 2, 2]
输出: 2

挑战
要求时间复杂度为O(n)，空间复杂度为O(1)

```python
nums = [1, 1, 1, 2, 2, 2, 2,2]

{nums.count(i):i for i in set(nums)}[max({nums.count(i):i for i in set(nums)})]
```




    2
