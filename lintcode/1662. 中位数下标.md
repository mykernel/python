给出一个含有 n 个互不相等整数的无序数组，找到其中中位数的下标。下标从 0 开始。

中位数是指这些数排序后最中间的数。

若 n 为偶数，则中位数是数组排序后的第 n/2 个数。


1 <= n <= 100000
所有数都各不相同

样例
样例 1：

输入：[4,5,1,2,3]
输出：4

样例 2：

输入：[7,9,4,5]
输出：31。 找到中位数
2。 找到下标

```python
lst = [4,5,1,2,3]
lst1 = sorted(lst)

if len(lst1) /2 == 0:
     print(lst.index(lst1[len(lst1)/2]))
    
else:
     print(lst.index(lst1[int((len(lst1)-1)/2)]))
```
