对于一个给定的 source 字符串和一个 target 字符串，你应该在 source 字符串中找出 target 字符串出现的第一个位置(从0开始)。如果不存在，则返回 -1。

注意：
1. 考虑"" 的问题
2. 从多到少思考

```python
class Solution:
    """
    @param source: 
    @param target: 
    @return: return the index
    """
    def strStr(self, source, target):
        # Write your code here
        if source == "":
            if target =="":
                return 0
            else:
                return -1 
                
        if target == "":
            if source =="":
                return -1
            else:
                return 0        
    
            
        try:
            source.index(target)
        except ValueError:
            print(-1)
        
        return source.find(target)

```
