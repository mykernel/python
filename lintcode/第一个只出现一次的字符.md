给出一个字符串，找出第一个只出现一次的字符。

说明：1.返回第一个，set方法不能使用。
2. 考虑时间复杂的，需要借助有序字典

```python
str = 'asdaswrwdasdasfdsggrgrhtjhyp'
import collections

d = collections.OrderedDict(zip(str,[0]*len(str)))


for i in d.keys():
    d[i] = str.count(i)
    
for k,v in d.items():
    if v == 1:
        print(k)
        break


```
