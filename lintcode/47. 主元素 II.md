给定一个整型数组，找到主元素，它在数组中的出现次数严格大于数组元素个数的三分之一。


数组中只有唯一的主元素
 
样例
例1:

输入: [99,2,99,2,99,3,3], 
输出: 99.
例2:

输入: [1, 2, 1, 2, 1, 3, 3], 
输出: 1.

要求时间复杂度为O(n)，空间复杂度为O(1)。

```python
nums  = [1, 2, 1, 2, 1, 3, 3]
```


```python

{nums.count(i):i for i in set(nums)}[max({nums.count(i):i for i in set(nums)})]
```




    1
