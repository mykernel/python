要求：
求两个字符串的最大公共子串

提示：
两种方案
1. 传统比较方案
2. 矩阵算法

```python
def findit(str1,str2):
    count = 0 
    length = len(str1)
    
    for sublen in range(length,0,-1):
        for start_index in range(0,length-sublen+1):
            substr = str1[start_index:start_index+sublen]
            count += 1 
            
            if str2.find(substr) > -1:
                print(count)
                return substr
```


```python
s1 = 'abcdfg'
s2 = 'defgabcdefasd2rf'

findit(s1,s2)
```
