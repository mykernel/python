给出两个数字 A 和 B，找出 A 与 B 之间所有的数（包括 A 和 B），使得这些数满足是 3 的倍数，返回这些数的和。

```python
class Solution:
    """
    @param A: The A
    @param B: The B
    @return: Returns the sum of all qualified numbers
    """
    def getSum(self, A, B):
        # Write your code here
        lst = [A,B]
        lst1 = []
        if A==B and A % 3 ==0:
            lst1 = [A]
        else:
            for i in range(A,B):
                if i % 3 == 0 :
                    lst1.append(i)

        return sum(lst1)
```
