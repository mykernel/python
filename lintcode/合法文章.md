给定一篇由大写字母、小写字母、逗号、句号组成的文章，求使文章不合法的字母数。
文章不合法有2种情况：
1.句子的第一个字母用了小写。
2.不是单词的首字母用了大写。

```python
s="This won iz correkt. it has, No Mistakes et Oll. But there are two BIG mistakes in this sentence. and here is one more."

#s = "CsXwLzffsbRnGOKGnwhmSiT.BOBFSKIFufrRJ lfmrhFIPsDyX"
count = 0
lst = []
if s.endswith('.'):
    s_new = s
else:
    s_new = s + '.'

for i in s_new.split('.')[:-1]:
        if not i.strip()[0].isupper() :
            count += 1
            
for i in s_new.split('.')[:-1]:
#   print(i)
    for i in (i.strip().split(' ')):
        lst.append(i[1:])

str_islower = ' '.join(lst)
# print(str_islower)

for  i in str_islower:
    if i.isupper():
        count+= 1 
        
print(count)


```
