给出一个有n个整数的数组S，在S中找到三个整数a, b, c，找到所有使得a + b + c = 0的三元组。

在三元组(a, b, c)，要求a <= b <= c。

结果不能包含重复的三元组。
  
样例
例1:

输入:[2,7,11,15]
输出:[]
例2:

输入:[-1,0,1,2,-1,-4]
输出:[[-1, 0, 1],[-1, -1, 2]]

```python
ll = [-1,0,1,2,-1,-4,2]
tmp = []
new = []

for i,v in enumerate(ll):
    for j in ll[i+1:]:
        # print(ll)
        for m in ll[i+2:]:   
            if v+j+m == 0: 
                tmp.append(sorted([v,j,m]))

for i in tmp:
    if i not in new:
        new.append(i)
print(new)

            
        
                
#print(new)
```
